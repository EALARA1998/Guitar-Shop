# Guitar Shop

Basic guitar shop web application built as a first React project to practice component structure and rendering dynamic data.

The application displays a simple catalog of guitars and focuses on understanding core React concepts.

## Features
- Guitar catalog display
- Reusable React components
- Basic application structure
- Simple and clean UI

## Tech Stack
- Frontend: React, TypeScript
- Build Tool: Vite
